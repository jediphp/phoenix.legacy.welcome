Author Flow Local Images
========================

All images used by `AuthorSlide*` are stored locally in this folder.

How to replace an image
-----------------------

1. Keep the same filename in the same `slide-XX` directory.
2. Replace file contents with a new image.
3. No code changes are required if path and filename stay the same.
